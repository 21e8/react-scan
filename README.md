# react-scan

Scan your React app for renders

## Install

**Install via npm**

```bash
npm install react-scan
```

```js
import 'react-scan/auto';
```

**Install via script tag**

```html
<script src="https://unpkg.com/react-scan/auto.global.js" />
```
